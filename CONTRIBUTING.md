# Contributing

Thanks for checking this out! If you want to contribute, here's how.

## Found a Bug?

Open an issue and include:
- What you expected to happen
- What actually happened
- Steps to reproduce it
- Your environment (PHP version, OS, browser)

## Want to Add Something?

Cool! Here's the process:

1. Fork the repo
2. Create a branch: `git checkout -b my-feature`
3. Make your changes
4. Test everything works
5. Commit: `git commit -m 'Added cool feature'`
6. Push: `git push origin my-feature`
7. Open a pull request

## Code Style

Keep it simple:

**PHP:**
- Use clear variable names
- Comment complex stuff
- Always use prepared statements for database queries
- Validate user input

**CSS:**
- Keep selectors simple
- Group related properties
- Use the existing color variables

**JavaScript:**
- Keep it vanilla (no jQuery needed)
- Use meaningful function names
- Comment anything tricky

## Testing

Before submitting:
- Test in Chrome, Firefox, and Safari
- Check mobile view
- Make sure forms still work
- No PHP errors

## Questions?

Just open an issue. I'm happy to help.
